Cú pháp của media querry:
        @media screen and (condition) {
            selector1 {
                // code css cho selector1
            }
        }

        Kỹ thuật media querry dùng để xử lý các màn hình có kích cỡ khác nhau
        như là smartphone, máy tính bảng ipad, màn hình laptop, màn hình desktop
